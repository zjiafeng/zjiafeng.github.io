hexo使用vscode编写乱码记录

发表于 2020-06-05 分类于 hexo, vscode Valine：
本文字数： 195 阅读时长 ≈ 1 分钟

      

前两天使用hexo记录一篇博客，打开后出现乱码的问题，记录下解决方法。

  1. 将文件目录下的 _config.yml 保存为utf-8格式(或者unicode格式)
  2. 将 新添加的md文件也保存为utf-8 格式(或者Unicode 格式)
  3. 重新启动hexo clean && hexo generate && hexo s
